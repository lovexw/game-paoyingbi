# 🪙 抛硬币游戏 - 娱乐性质

一个功能完整的抛硬币游戏，具有优雅的动画效果、真随机算法和完整的统计功能。

## ✨ 功能特点

### 🎮 三种游戏模式

1. **测试模式**
   - 不消耗真实资金
   - 可测试 1-100 局
   - 记录测试数据供参考

2. **正式游戏**
   - 单次下注投注
   - 选择正面或反面
   - 实时结算盈亏

3. **连续下注**
   - 批量自动投注
   - 最多下注至余额用尽
   - 显示实时进度和净收益

### 💰 游戏规则

- 初始资金: **$10,000**
- 最小下注: **$100** (必须是100的倍数)
- 每局手续费: **$5**
- 赢了获得: **$200** (手续费不返还)
- 输了扣除: 下注金额 + 手续费

### 🎲 真随机保证

使用 **Web Crypto API** (`crypto.getRandomValues()`) 提供加密级别的真随机数生成，确保:
- 完全不可预测
- 加密安全级别
- 无法被操纵
- 每次结果独立

### 📊 统计功能

- 测试局数和正面率
- 正式局数和正面率
- 基于历史数据的概率预测
- 最近 20 条游戏记录
- 实时余额显示

### 💾 数据持久化

所有数据保存在浏览器本地存储 (localStorage)，包括:
- 用户余额
- 测试统计
- 正式游戏统计
- 历史记录 (最多保存 100 条)

### 🎨 界面特点

- 响应式设计，适配各种屏幕
- 流畅的硬币翻转动画
- 优雅的深色主题
- 平滑的过渡效果
- 直观的数据可视化

## 🎯 核心算法

### 真随机生成

```javascript
trueRandom() {
    const array = new Uint32Array(1);
    crypto.getRandomValues(array);
    return array[0] % 2 === 0 ? 'heads' : 'tails';
}
```

### 概率计算

基于历史数据计算下次出现正面/反面的概率，优先使用正式游戏数据，若无则使用测试数据，都没有则默认 50/50。
