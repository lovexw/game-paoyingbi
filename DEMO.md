# 🎮 游戏演示说明

## 如何运行

1. 在浏览器中打开 `index.html` 文件
2. 无需安装任何依赖或启动服务器
3. 直接开始游戏！

## 功能演示

### 1. 测试模式 🧪

**目的**: 在不消耗资金的情况下测试游戏，了解硬币的随机性

**操作步骤**:
1. 默认就在"测试模式"
2. 输入想测试的局数 (1-100)
3. 点击"开始测试"
4. 观察硬币翻转动画
5. 查看左侧统计数据更新

**效果**:
- 硬币会翻转并显示结果（正面或反面）
- 测试局数统计会增加
- 测试正面率会实时更新
- 不会消耗余额

### 2. 正式游戏 🎮

**目的**: 使用真实游戏资金进行投注

**操作步骤**:
1. 点击"正式游戏"切换模式
2. 输入下注金额 (100的倍数，最小100)
3. 选择"正面"或"反面"
4. 等待硬币翻转
5. 查看结果和盈亏

**规则**:
- 下注: $100 (或100的倍数)
- 手续费: $5
- 猜对: 获得 $200 (净利润 $95)
- 猜错: 失去 $105 (下注金额 + 手续费)

### 3. 连续下注 🔄

**目的**: 批量自动下注，测试长期收益

**操作步骤**:
1. 点击"连续下注"切换模式
2. 输入单次下注金额
3. 输入连续局数 (系统会根据余额自动计算最大局数)
4. 选择"全部押正面"或"全部押反面"
5. 观看自动执行
6. 查看最终净收益

**特点**:
- 自动执行多局
- 实时显示进度条
- 显示当前局数和净收益
- 余额实时更新

## 真随机保证 🎲

本游戏使用 `crypto.getRandomValues()` API:

```javascript
trueRandom() {
    const array = new Uint32Array(1);
    crypto.getRandomValues(array);
    return array[0] % 2 === 0 ? 'heads' : 'tails';
}
```

**特点**:
- ✅ 加密级别的随机数生成器
- ✅ 无法预测
- ✅ 无法操纵
- ✅ 每次调用完全独立
- ✅ 符合密码学安全标准

**好处**:

- 跨标签页数据同步
- 完全本地存储，隐私安全

## 建议游戏流程 🎯

### 新手流程

1. **了解游戏** (5分钟)
   - 阅读右侧游戏规则
   - 查看界面各部分功能

2. **测试体验** (10分钟)
   - 测试1局，观察动画
   - 测试10局，查看统计
   - 测试100局，验证随机性

3. **小额试玩** (15分钟)
   - 切换到正式游戏
   - 下注$100试玩几局
   - 体验输赢感觉

4. **策略尝试** (20分钟)
   - 根据统计数据选择
   - 尝试连续下注
   - 观察长期收益

### 进阶玩法

1. **数据分析流派**
   - 先测试100局收集数据
   - 分析正反面概率
   - 根据概率下注

2. **稳健投资流派**
   - 固定下注$100
   - 不追逐损失
   - 设定止盈止损点

3. **激进赌徒流派**
   - 大额下注
   - 连续下注
   - 追求快速翻倍
